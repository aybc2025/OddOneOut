# 🎯 משחק "מה יוצא דופן?" (Odd One Out)

משחק חשיבה אינטראקטיבי לילדים בגילאי 4-6, המבוסס על זיהוי הפריט היוצא דופן מתוך קבוצה.

## ✨ תכונות

- 🎨 30 חידות מגוונות
- 📱 PWA - עובד במצב offline
- 🌐 תמיכה מלאה בעברית ו-RTL
- 🎭 100+ אייקוני SVG מעוצבים
- 📲 מותאם למובייל וטאבלט
- 🔄 ערבוב אוטומטי של החידות
- ✅ משוב ויזואלי מיידי
- 🚀 ללא צורך בשרת

## 🚀 התקנה והרצה

### הרצה מקומית

1. פתח את הפרויקט:
```bash
cd odd-one-out-game
```

2. הפעל שרת HTTP פשוט:
```bash
# Python 3
python -m http.server 8000

# או Node.js
npx http-server -p 8000
```

3. פתח בדפדפן:
```
http://localhost:8000
```

### התקנה כ-PWA

1. פתח את המשחק בדפדפן נייד (Chrome/Safari)
2. לחץ על "הוסף למסך הבית"
3. המשחק יפעל כאפליקציה עצמאית

## 📁 מבנה הפרויקט

```
odd-one-out-game/
├── index.html              # HTML ראשי
├── styles.css              # עיצוב ואנימציות
├── main.js                 # לוגיקת המשחק
├── manifest.json           # הגדרות PWA
├── service-worker.js       # Service Worker לתמיכה offline
├── data/
│   ├── riddles.json        # 30 החידות
│   └── svg-map.json        # מיפוי SVG icons
└── assets/
    ├── svg/                # 100+ אייקוני SVG
    ├── icon.svg            # אייקון PWA (SVG)
    ├── icon-192.png        # אייקון 192x192
    └── icon-512.png        # אייקון 512x512
```

## 🎮 איך זה עובד?

1. **טעינה**: המשחק טוען את כל החידות וה-SVG icons
2. **הצגה**: מוצגת חידה אקראית עם 4-6 פריטים
3. **בחירה**: הילד לוחץ על הפריט שנראה לו שונה
4. **משוב**:
   - ✅ תשובה נכונה → מעבר לחידה הבאה
   - ❌ תשובה שגויה → הסבר והזדמנות לנסות שוב
5. **המשך**: אחרי כל החידות, המשחק מערבב ומתחיל מחדש

## 🤝 תרומה

רוצים להוסיף חידות? כך עושים זאת:

1. ערכו את `data/riddles.json`
2. הוסיפו SVG icons חדשים ל-`assets/svg/`
3. עדכנו את `data/svg-map.json`

פורמט חידה:
```json
{
  "id": "q31",
  "items": [
    {"type": "item1"},
    {"type": "item2"},
    {"type": "item3"},
    {"type": "odd-one"}
  ],
  "correctIndex": 3,
  "explanation": "הסבר בעברית"
}
```
