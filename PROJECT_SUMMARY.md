## 🚀 איך להריץ?

```bash
cd odd-one-out-game
python -m http.server 8000
# פתח http://localhost:8000
```

## ⚠️ לפני הפריסה

1. **המר אייקונים ל-PNG:**
   ```bash
   cd assets
   convert -background none icon.svg -resize 192x192 icon-192.png
   convert -background none icon.svg -resize 512x512 icon-512.png
   ```

2. **בדוק שהכל עובד:**
   - ✅ כל ה-SVG נטענים
   - ✅ החידות מוצגות

   - או כל שרת סטטי
